# Agent Memory Architectures

Research and production memory systems for AI agents. Each architecture is a self-contained module with its own deployment configuration, test suite, and documentation.

## Architectures

| Name | Description | Status |
|------|-------------|--------|

## Organization

This is a monorepo. Each architecture lives in its own directory with independent dependencies, tests, and deployment configs. Shared patterns emerge from practice, not premature abstraction.
